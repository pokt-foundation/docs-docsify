＃ 欢迎

Pocket Network 是 Web3 节点基础设施的 TCP/IP——一种多链中继协议，它激励 RPC 节点为 DApp 及其用户提供不可阻挡的 Web3 访问。

Pocket 支持数十个区块链，并且一直在添加更多。

[**查看支持的区块链列表**](supported-blockchains.md)
